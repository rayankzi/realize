- **Insight 1:** System design remains a uniquely human skill — making trade-off decisions about optimization, scalability, and what breaks under load is something AI can't reliably do.

- **Insight 3:** Code security is critical because working code ≠ safe code — vulnerabilities, abuse cases, and exploit paths still require human judgment to identify.
- **Insight 4:** Code reviewing is becoming even more important as AI-generated code proliferates — engineers need to read critically, give quality feedback, and catch issues.
