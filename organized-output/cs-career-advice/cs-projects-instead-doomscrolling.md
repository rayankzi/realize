# CS Projects to Build Instead of Doomscrolling

**Topic/Category:** Tech / Computer Science / Programming Projects
**Primary Goal of the Post:** To inspire CS students and aspiring developers to spend their free time building impressive projects by recommending specific YouTube tutorials for each one.

---

## Executive Summary

This carousel post by @christyoverflow curates a list of six hands-on computer science projects that viewers can build by following along with popular YouTube tutorials. The projects span AI/ML, web development, and systems programming — all designed to be more productive alternatives to mindless scrolling. Each slide pairs a project idea with a specific, well-known YouTube video tutorial to follow, lowering the barrier to getting started.

## Key Insights & Main Ideas

- **Learn by building:** The post emphasizes project-based learning over passive consumption, framing coding projects as the productive alternative to doomscrolling.
- **Curated tutorial pairing:** Each project comes with a specific YouTube tutorial recommendation, removing the friction of finding resources.
- **Range of difficulty and domain:** Projects range from beginner-friendly (chat app) to advanced (building GPT from scratch), covering AI, ML, web dev, and information retrieval.
- **Resume-worthy projects:** The caption hints at resume value, offering a follow-up guide with "10 more CS projects and resume bullet points for each."
- **Popular, vetted resources:** The recommended tutorials are from well-known creators (Andrej Karpathy, Web Dev Simplified, etc.) with high view counts, adding credibility.
