### Full-Stack Projects (1–4)

1. **Real-Time Collaborative Notes App** — A Google Docs lite clone with live multi-user editing via WebSockets. Tech: Next.js, WebSockets, Supabase.
2. **Job Application Tracker** — Full CRUD app with authentication, status tracking, and email reminders. Tech: Next.js, PostgreSQL, Resend.

5. **RAG Chatbot on Your Own Docs** — Upload PDFs and chat with them, demonstrating embeddings, chunking, and retrieval skills. Tech: Python, LangChain, Pinecone or Supabase, OpenAI.
6. **Fraud Detection System** — Work with real imbalanced data, compare models, and deploy a Streamlit demo. Tech: Python, scikit-learn, FastAPI, Streamlit.
7. **AI Resume Screener** — Upload a resume and job description, receive a fit score with reasoning. Tech: Next.js, OpenAI API, Supabase.

### Backend / Infrastructure Projects (8–10)

8. **Rate Limiter from Scratch** — Implement token bucket and sliding window algorithms as reusable middleware. Tech: Python or Go, Redis.
9. **Mini Load Balancer** — Round-robin HTTP load balancer with health checks across local servers. Tech: Go.
10. **REST API with Auth + Role-Based Access** — Full auth system with JWT, refresh tokens, and admin vs. user permissions. Tech: Node.js or FastAPI, PostgreSQL, JWT.
