# Data Science 4-Year University Roadmap: From Foundations to Production AI

**Topic/Category:** Tech / Education / Data Science Career Guide
**Primary Goal of the Post:** To provide data science students with a comprehensive, year-by-year university roadmap covering classes, self-study skills, and project ideas that progressively build from math fundamentals to production-level AI systems.

---

## Executive Summary

This carousel by @tinahelloworld (a CS senior, incoming engineer at Apple, and 5x intern) lays out a structured 4-year data science university guide. The roadmap progresses from mastering math and programming fundamentals in Year 1, through machine learning and internship preparation in Year 2, advanced ML and deep learning in Year 3, to specialization and production-level AI systems in Year 4. Each year includes recommended classes, self-directed learning goals, and concrete project ideas to build a competitive portfolio.

## Key Insights & Main Ideas

- **Insight 1:** The roadmap emphasizes parallel tracks -- formal coursework alone is insufficient. Each year pairs university classes with self-directed skills (tools, frameworks, competitions) and portfolio projects.
- **Insight 2:** Internship timing is critical -- the guide advises applying early (Aug-Oct of Year 2) and targeting top DS/ML internships by Year 3, reinforcing that career preparation should start well before senior year.
- **Insight 3:** The progression follows a clear trajectory: math foundations -> classical ML -> deep learning/NLP -> MLOps/production deployment, mirroring real industry expectations for data science roles.
- **Insight 4:** GitHub portfolio building starts from Year 1, emphasizing that consistent contributions and visible project work are essential from the very beginning (shown via the GitHub contribution graph on the cover slide).
- **Insight 5:** Year 4 introduces career specialization -- ML engineer, data scientist, analytics, or AI -- signaling that students should choose a lane rather than staying generalist.

## Detailed Breakdown / Step-by-Step

- **Slide 1 (Cover):** Title card -- "Data Science 4-Year University Guide / DS Survival Guide for Success." Features a mirror selfie in an ornate building lobby with a GitHub contribution graph overlay, visually linking consistent coding activity to success.

- **Slide 2 (Year 1) -- Master Math + Intro Programming Fundamentals:**
  - **Classes:** Programming 1 & 2 (Python), Calculus 1 & 2, Linear Algebra, Intro Statistics/Probability, General Education
  - **Learn & Do:** Python for data (numpy, pandas, matplotlib), Jupyter notebooks & exploratory analysis, basic SQL, data cleaning & visualization, create a GitHub portfolio
  - **Projects:** Exploratory data analysis on a public dataset; Movie recommendation system (collaborative filtering/similarity)

- **Slide 3 (Year 2) -- Build ML Fundamentals & Land First Internships:**
  - **Classes:** Data Structures & Algorithms, Probability & Mathematical Statistics, Database Systems (SQL + data models), Machine Learning fundamentals, Data Visualization
  - **Learn & Do:** Regression & classification models, model evaluation (accuracy, precision, recall), build strong ML portfolio projects, apply to internships early (Aug-Oct), participate in Kaggle competitions
  - **Projects:** Housing price prediction (regression + feature engineering); Fraud detection classifier (random forest/gradient boosting); Interactive data dashboard (Streamlit/Tableau/PowerBI)

- **Slide 4 (Year 3) -- Advanced ML + High-Impact Internships:**
  - **Classes:** Advanced Machine Learning, Deep Learning/Neural Networks, Big Data Systems (Spark/distributed data), Data Mining, Natural Language Processing
  - **Learn & Do:** Deep learning (PyTorch or TensorFlow), experiment tracking (W&B/MLflow), large dataset processing (Spark/PySpark), system design for ML pipelines, apply to top DS/ML internships early
  - **Projects:** Deep learning image classifier (CNN using PyTorch); NLP sentiment analysis system (transformers/LLM fine-tuning); End-to-end ML pipeline (data ingestion -> training -> deployment)

- **Slide 5 (Year 4) -- Specialize & Build Production-Level AI Systems:**
  - **Classes:** Senior Capstone/Data Science Project, Reinforcement Learning or Advanced ML, Distributed Data Systems, AI Ethics/Responsible AI
  - **Learn & Do:** Deploy ML models (FastAPI + Docker), production ML pipelines (MLOps), publish projects & technical blogs, apply to DS/ML roles early, choose specialization (ML engineer, data scientist, analytics, AI)
  - **Projects:** Production ML API (real-time prediction service); Large-scale data pipeline (Spark + cloud infrastructure); Reverse-engineer real datasets and build predictive models

- **Slide 6 (Outro):** Personal credentials -- "sincerely, a CS senior, incoming eng @ apple, 5x intern, and passionate developer." Promotes coding, tech, CS, and university content with a follow CTA.
